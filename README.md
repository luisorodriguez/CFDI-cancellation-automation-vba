# CFDI-cancellation-automation-vba
Excel VBA automation for CFDI cancellation workflow

# Problem
The original workflow required:
- Manual download of cloud spreadsheets
- Manual consolidation of multiple files
- Data validation by column position
- Manual layout construction
- Manual incident creation
- Manual email drafting
Execution time: ~45 minutes per run
High risk of manual error.

# Solution
A modular VBA automation that:
- Consolidates multiple source files
- Dynamically processes structured data
- Generates standardized output layouts
- Automates incident script generation
- Drafts operational communication templates
- Reduces execution time by ~70%

# Technical Concepts Demonstrated
- Multi-workbook handling
- Dynamic row detection
- Data transformation
- Modular VBA architecture
- Environment-based path handling
- Secure placeholder usage for sensitive data
